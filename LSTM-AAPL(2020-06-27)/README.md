# Predicting Stock Prices Using An LSTM Model

![Plot](https://github.com/jinwei-ang/Stock-Market-Analysis-Mini-Projects/blob/main/LSTM-AAPL(2020-06-27)/LSTM-AAPL(2020-06-27).png)
